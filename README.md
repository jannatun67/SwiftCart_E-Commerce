1) What is the difference between null and undefined?
Ans: যখন কোনো ভ্যারিয়েবল declare করা হয় কিন্তু value assign করা হয় না, তখন তার মান হয় undefined।  null মানে ইচ্ছাকৃতভাবে খালি  value দেওয়া হয়েছে।


2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Ans: একটি array এর প্রতিটি element এর উপর কাজ করে। নতুন একটি array return করে। forEach() array এর প্রতিটি element এর উপর কাজ করে। কিন্তু নতুন array return করে না।

3) What is the difference between == and ===?
Ans: == type check করে না। And === type এবং value দুটোই check করে।

4) What is the significance of async/await in fetching API data?
Ans: asynchronous code কে synchronous style এ লিখতে সাহায্য করে। code readable ও clean হয়।  error handle করা সহজ হয় (try...catch দিয়ে)।
